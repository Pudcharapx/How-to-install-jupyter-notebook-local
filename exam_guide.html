Data Cleaning & Analysis

อธิบายเจาะลึกแบบบรรทัดต่อบรรทัดจากไฟล์ ติวDC.ipynb เพื่อให้เข้าใจหลักการวิเคราะห์ข้อมูล จัดการข้อมูลซ้ำ จัดการข้อมูลสูญหาย และการประมวลผลเกรดนักศึกษาอย่างเป็นขั้นตอน

01

การอ่านไฟล์ Excel หลาย Sheet

ในข้อสอบ ข้อมูลมักจะมาในรูปแบบไฟล์ Excel 1 ไฟล์แต่มีหลาย Sheet เราจะใช้ pd.read_excel ดึงข้อมูลออกมา

[1]:
import pandas as pd # อ่านข้อมูลจากไฟล์ Excel โดยดึงมาทุก Sheet ด้วย sheet_name=None reg_sheets = pd.read_excel("RegistrationName.xlsx", sheet_name=None) score_sheets = pd.read_excel("StudentScores.xlsx", sheet_name=None) # แยกข้อมูลจากแต่ละ Sheet ออกมาเป็น DataFrame แต่ละตัว reg_df = reg_sheets["NameList"] hw_df = score_sheets["Homework"] quiz1_df = score_sheets["Quiz1"] quiz2_df = score_sheets["Quiz2"] mt_df = score_sheets["MidtermExam"] fn_df = score_sheets["FinalExam"]
คำอธิบายโค้ด:
  • import pandas as pd: นำเข้าไลบรารี Pandas และตั้งชื่อเล่นว่า pd เพื่อใช้ในการจัดการตารางข้อมูล
  • sheet_name=None: เป็นคำสั่งสำคัญที่บอกให้ Pandas ดึงข้อมูล ทุก Sheet ในไฟล์นั้นขึ้นมา โดยเก็บอยู่ในรูปแบบ Dictionary (ชื่อ Sheet เป็น Key, ข้อมูลเป็น Value)
  • reg_sheets["NameList"]: คือการดึงข้อมูลเฉพาะ Sheet ที่ชื่อว่า "NameList" ออกมาเก็บไว้ในตัวแปร reg_df เพื่อให้ง่ายต่อการใช้งานในขั้นตอนต่อไป
02

การสำรวจและลบข้อมูลซ้ำ (Duplicated Rows)

ข้อมูลดิบมักจะมีข้อผิดพลาด เช่น มีการกรอกข้อมูลซ้ำ เราต้องเช็คและลบทิ้งก่อนวิเคราะห์

[2]:
print("จำนวนแถวซ้ำ:", hw_df.duplicated().sum()) # ลบข้อมูลที่ซ้ำกันออกไป hw_df = hw_df.drop_duplicates() print("จำนวนแถวหลังลบ:", len(hw_df))
คำอธิบายโค้ด:
  • hw_df.duplicated(): จะตรวจสอบทุกแถวว่ามีแถวไหนที่ข้อมูลซ้ำกับแถวก่อนหน้าหรือไม่ จะคืนค่าเป็น True/False
  • .sum(): นำมาใช้ต่อท้ายเพื่อหาว่ามีแถวซ้ำทั้งหมดกี่แถว (True มีค่าเป็น 1)
  • drop_duplicates(): ทำการลบแถวที่มีข้อมูลซ้ำกันทุกประการออกไป (จะเก็บไว้เพียงแถวแรกสุดที่เจอเท่านั้น)
03

การจัดการ Missing Value (NA) และแปลงชนิดข้อมูล

ข้อมูลอาจจะมีช่องว่าง (NaN) หรือพิมพ์ข้อความแปลกๆ ลงในช่องคะแนน เช่น "-" หรือ "none" ซึ่งเราคำนวณไม่ได้

[3]:
# เช็คจำนวนค่าว่างในแต่ละคอลัมน์คิดเป็นเปอร์เซ็นต์ print(hw_df.isnull().sum() / len(hw_df) * 100) # เติมค่าว่างด้วย 0 ทั้งหมด hw_df.fillna(0, inplace=True) quiz1_df.fillna(0, inplace=True) # แปลงชนิดข้อมูลให้เป็นตัวเลข กรณีที่มี text ปนมาในคอลัมน์คะแนน hw_df["HW4_10"] = pd.to_numeric(hw_df["HW4_10"], errors="coerce").fillna(0) quiz1_df["Quiz1_10"] = pd.to_numeric(quiz1_df["Quiz1_10"], errors="coerce").fillna(0)
คำอธิบายโค้ด:
  • .isnull().sum(): เช็คว่าแต่ละคอลัมน์มีค่าว่าง (NaN) กี่ตัว จากนั้นหารด้วยจำนวนแถวทั้งหมด (len) คูณ 100 เพื่อดูเป็นเปอร์เซ็นต์
  • fillna(0, inplace=True): แปลงช่องที่ว่าง (ไม่มีข้อมูล) ให้เป็นเลข 0 ทั้งหมด โดย inplace=True หมายถึงให้แก้ไขทับข้อมูลเดิมใน DataFrame ทันที
  • pd.to_numeric(..., errors="coerce"): พยายามแปลงข้อมูลในคอลัมน์ให้เป็นตัวเลข (Float) แต่ถ้าแปลงไม่ได้ (เช่น พิมพ์มาว่า "ขาดสอบ" หรือ "-") ตัว errors="coerce" จะทำให้ค่านั้นกลายเป็น NaN แทน จากนั้นเราจึงใช้ .fillna(0) ต่อท้ายเพื่อเปลี่ยน NaN ให้เป็น 0
04

การรวมข้อมูล (Merge/Join)

เมื่อเราทำความสะอาดข้อมูลแยกตาม Sheet เสร็จแล้ว เราจะต้องเอาทุก Sheet มารวมเป็นตารางเดียว เพื่อรวมคะแนน

[4]:
# รวมข้อมูลทีละตารางโดยใช้ SID เป็นจุดเชื่อม merged = reg_df.merge(hw_df[["SID","HW1_10","HW2_5","HW3_15","HW4_10"]], on="SID") merged = merged.merge(quiz1_df[["SID","Quiz1_10"]], on="SID") merged = merged.merge(quiz2_df[["SID","Quiz2_10"]], on="SID") merged = merged.merge(mt_df[["SID","MT_25"]], on="SID") merged = merged.merge(fn_df[["SID","FN_25"]], on="SID") # เลือกจัดเรียงเฉพาะคอลัมน์ที่เราจะใช้งาน merged = merged[["SID","Name","Email","Section","HW1_10","HW2_5","HW3_15", "HW4_10","Quiz1_10","Quiz2_10","MT_25","FN_25"]] # ตั้งให้ SID เป็น Index (หัวแถว) ของตารางแทนที่เลข 0,1,2... merged = merged.set_index("SID")
คำอธิบายโค้ด:
  • .merge(ตารางขวา, on="SID"): ทำการเชื่อมตารางซ้ายและขวาเข้าด้วยกัน โดยดูจากคอลัมน์ SID (รหัสนักศึกษา) ถ้ารหัสตรงกันก็จะนำข้อมูลมาต่อกันในแนวนอน
  • hw_df[["SID", "HW1_10", ...]]: การใส่ List ซ้อนด้านใน [[...]] คือการ "เลือกเฉพาะบางคอลัมน์" จากตาราง เพื่อไม่ให้ตารางที่ได้จากการ merge มีคอลัมน์ขยะติดมาด้วย
  • set_index("SID"): เปลี่ยนให้รหัสนักศึกษากลายเป็นตัวอ้างอิงของแต่ละแถว (Index) ซึ่งจะมีประโยชน์มากตอนเราดูข้อมูลเป็นรายบุคคล
05

การสร้างคอลัมน์คำนวณคะแนนรวม

คณิตศาสตร์พื้นฐาน นำแต่ละคอลัมน์มาบวกกันเพื่อหาคะแนนเก็บและคะแนนรวมทั้งหมด

[5]:
# สร้างคอลัมน์ใหม่มารองรับผลบวก merged["Total HW"] = merged["HW1_10"] + merged["HW2_5"] + merged["HW3_15"] + merged["HW4_10"] merged["Total QZ"] = merged["Quiz1_10"] + merged["Quiz2_10"] # คำนวณคะแนนรวมทั้งหมด merged["Total Scores"] = merged["Total HW"] + merged["Total QZ"] + merged["MT_25"] + merged["FN_25"]
คำอธิบายโค้ด:
  • merged["ชื่อคอลัมน์ใหม่"] = ...: ใน Pandas เราสามารถสร้างคอลัมน์ใหม่ได้ง่ายๆ ด้วยการกำหนดชื่อที่ไม่มีอยู่จริง และจับเท่ากับสมการที่เราต้องการ คล้ายกับการสร้างสูตรใน Excel ทั่วไป
  • เราสามารถนำ Series (คอลัมน์) มาบวกกันได้โดยตรง Pandas จะทำการบวกข้อมูลแถวต่อแถว (Row by Row) ให้อัตโนมัติ
06

การจัดเรียงและคัดกรองข้อมูล

เมื่อได้คะแนนรวมแล้ว เรามักจะต้องหาว่าใครได้คะแนนสูงสุด หรือกรองคนสอบผ่าน

[6]:
# จัดเรียงคะแนนรวมจากมากไปน้อย merged = merged.sort_values(by="Total Scores", ascending=False) # คัดกรองเอาเฉพาะคนที่ได้คะแนน >= 60 ขึ้นไป df_60up = merged.loc[merged["Total Scores"] >= 60] # จัดเรียงอีกครั้งจากน้อยไปมาก df_60up = df_60up.sort_values(by="Total Scores") # เลือกข้อมูล 5 แถวแรก (Top 5) df_top5 = df_60up.iloc[0:5][["Name", "Email", "Section", "Total Scores"]]
คำอธิบายโค้ด:
  • sort_values(by="...", ascending=False): เรียงลำดับตารางโดยดูจากคอลัมน์เป้าหมาย ค่า ascending=False หมายถึงเรียงจากมากไปน้อย (ค่า Default คือ True คือเรียงจากน้อยไปมาก)
  • merged.loc[...]: ใช้ในการคัดกรอง (Filter) ข้อมูลตามเงื่อนไขที่กำหนด ในที่นี้คือ merged["Total Scores"] >= 60
  • .iloc[0:5]: เลือกข้อมูลตามลำดับแถวที่ปรากฏในตาราง (Index Location) โดย 0:5 หมายถึงเอาตั้งแต่แถวที่ 0 ไปจนถึงแถวที่ 4 (รวม 5 แถว)
07

การสร้างฟังก์ชันตัดเกรด

หากเรามีเงื่อนไขที่ซับซ้อน เช่น การให้เกรด A B C D F เราสามารถเขียนฟังก์ชันของ Python แล้วนำไปใช้งานกับ Pandas ได้

[7]:
# นิยามฟังก์ชันตัดเกรดตามเงื่อนไข def grading(score): if score >= 80: return "A" elif score >= 70: return "B" elif score >= 60: return "C" elif score >= 50: return "D" else: return "F" # วนลูปส่งคะแนนทีละคนเข้าไปในฟังก์ชัน แล้วเก็บเกรดไว้ใน List grade_list = [] for score in merged["Total Scores"]: grade_list.append(grading(score)) # นำ List กลับไปสร้างเป็นคอลัมน์ใหม่ในตาราง merged["Grading"] = grade_list
คำอธิบายโค้ด:
  • def grading(score):: สร้างฟังก์ชันรับค่าคะแนนดิบ 1 ตัว และทำการเช็คเงื่อนไข if-elif-else เพื่อคืนค่าตัวอักษรเกรดกลับมา
  • for score in merged["Total Scores"]:: เป็นการดึงคะแนนของนักศึกษาออกมาทีละคน เพื่อโยนเข้าไปให้ฟังก์ชันคำนวณ และนำผลที่ได้ใส่ในกล่อง grade_list ด้วยคำสั่ง .append()
  • หลังจากวนลูปเสร็จ เราก็นำเอา grade_list ทั้งก้อน ยัดกลับเข้าไปเป็นคอลัมน์ใหม่ชื่อ Grading ในตารางหลัก
  • (เคล็ดลับ: ปกติเราสามารถใช้คำสั่ง merged["Total Scores"].apply(grading) ได้เลย ซึ่งสั้นและเร็วกว่าการใช้ for loop แต่ในคอร์สติวอาจจะสอนแบบ Loop เพื่อให้เห็นภาพการทำงานทีละขั้นตอนชัดเจน)
08

การหาค่าสถิติเบื้องต้น

สรุปข้อมูลภาพรวมของห้องเรียน เช่น คะแนนเฉลี่ย หรือคะแนนสูงสุด

[8]:
# ดูสถิติทั้งหมดทุกคอลัมน์ stats = merged.describe() # ดึงเฉพาะค่าที่ต้องการมาปริ้น print("Mean:", merged["Total Scores"].mean()) print("Max:", merged["Total Scores"].max()) print("Min:", merged["Total Scores"].min()) print("Standard Deviation:", merged["Total Scores"].std())
คำอธิบายโค้ด:
  • .describe(): เป็นฟังก์ชันสุดยอดของ Pandas ที่จะคำนวณสถิติพื้นฐาน (Count, Mean, Std, Min, Max, 25%, 50%, 75%) ของคอลัมน์ตัวเลขทุกคอลัมน์ในตารางให้โดยอัตโนมัติ
  • หากต้องการเรียกใช้ทีละค่าเฉพาะเจาะจง ก็สามารถเรียกผ่าน .mean() (ค่าเฉลี่ย), .max() (ค่ามากสุด), .min() (ค่าน้อยสุด), และ .std() (ส่วนเบี่ยงเบนมาตรฐาน) จากคอลัมน์นั้นๆ ได้เลย
