ติดตั้งและใช้งาน
Jupyter Notebook แบบ Local

คู่มือทีละขั้นตอนสำหรับรันทุกคำสั่งผ่าน PowerShell เพียงอย่างเดียว ตั้งแต่เช็ค Python ไปจนถึงเปิด Notebook ใช้งานจริงบน Desktop

PowerShell — Desktop\jupyter-project

PS C:\Users\golf\Desktop\jupyter-project> jupyter notebook

[I] Serving notebooks from local directory: C:\Users\golf\Desktop\jupyter-project

[I] Jupyter Notebook 7.2.0 is running at:

[I] http://localhost:8888/tree?token=a1b2c3...

[I] Use Control-C to stop this server.

STEP 01

เปิด PowerShell

  • กด Win + X แล้วเลือก Windows PowerShell หรือ Terminal
  • หรือกด Win พิมพ์ powershell แล้วกด Enter
STEP 02

ตรวจสอบว่ามี Python แล้วหรือยัง

พิมพ์คำสั่งด้านล่างเพื่อเช็คเวอร์ชัน Python ที่ติดตั้งอยู่

powershell
python --version
⚠️ สำคัญ: ถ้ายังไม่มี Python ให้โหลดจาก python.org แล้วต้องติ๊กช่อง "Add python.exe to PATH" ตอนติดตั้ง มิฉะนั้น PowerShell จะเรียกใช้ไม่ได้
STEP 03

อนุญาตให้รัน Script (ทำครั้งเดียว)

PowerShell จะบล็อกไฟล์ activate.ps1 ที่ venv ต้องใช้ ต้องปลดล็อกก่อนหนึ่งครั้ง

powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

ถ้าระบบถาม ให้พิมพ์ Y แล้วกด Enter

STEP 04

สร้างโฟลเดอร์โปรเจกต์บน Desktop

powershell
cd Desktop
mkdir jupyter-project
cd jupyter-project

โฟลเดอร์จะอยู่ที่ C:\Users\ชื่อเรา\Desktop\jupyter-project

STEP 05

สร้างและเปิดใช้งาน Virtual Environment

powershell
python -m venv venv
.\venv\Scripts\Activate.ps1

เมื่อ activate สำเร็จ จะเห็น (venv) ขึ้นด้านหน้า prompt

STEP 06

ติดตั้ง Jupyter Notebook

ขณะที่ (venv) ยัง active อยู่ ให้รัน:

powershell
pip install notebook

อยากได้อินเทอร์เฟซรุ่นใหม่ (JupyterLab) ให้เพิ่ม:

powershell
pip install jupyterlab
STEP 07

ติดตั้งไลบรารีที่ใช้บ่อย

ก่อนเริ่มเขียนโค้ดวิเคราะห์ข้อมูล ให้ติดตั้งไลบรารีพื้นฐานอย่าง numpy, pandas, matplotlib และ ipykernel ไว้ล่วงหน้า ทำได้ 2 วิธี:

วิธีที่ 1 — ติดตั้งจาก PowerShell (ก่อนเปิด notebook ขึ้นมา)

powershell
pip install numpy matplotlib pandas ipykernel openpyxl

วิธีที่ 2 — ติดตั้งจากภายใน Notebook โดยรันคำสั่งในเซลล์แรก (เครื่องหมาย ! ด้านหน้า คือการสั่งรันคำสั่ง shell จากใน notebook)

[2]:
! pip install numpy matplotlib pandas ipykernel openpyxl
Requirement already satisfied: numpy in c:\users\...\site-packages (2.3.5) ●●●
💡 ถ้าขึ้นข้อความ "Requirement already satisfied" แปลว่ามีไลบรารีนั้นติดตั้งอยู่แล้ว ไม่ต้องทำอะไรเพิ่ม สามารถ import ไปใช้งานได้เลย
STEP 08

เปิดใช้งาน Jupyter Notebook

powershell
jupyter notebook

ระบบจะแสดง URL เช่น http://localhost:8888/tree?token=... และเปิดเบราว์เซอร์อัตโนมัติ ถ้าไม่เปิดเอง ให้คัดลอก URL ไปวางในเบราว์เซอร์เอง

🛑ห้ามปิดหน้าต่าง PowerShell นี้ระหว่างใช้งาน — มันคือเซิร์ฟเวอร์ที่รันอยู่เบื้องหลัง
STEP 09

การใช้งานเบื้องต้น

สร้าง Notebook ใหม่: กด New มุมขวาบน → เลือก Python 3 (ipykernel)

ทดลอง import ไลบรารีที่ติดตั้งไว้ และสร้างตารางข้อมูลง่ายๆ ด้วย pandas:

[3]:
import numpy as np import pandas as pd import matplotlib
[6]:
x = ["1", "2", "3"] x = pd.DataFrame(x) x

รันโค้ดใน cell ด้วย Shift + Enter

คีย์ลัดหน้าที่

ปิดเซิร์ฟเวอร์: กลับไปที่ PowerShell กด Ctrl + C แล้วพิมพ์ y

STEP 10

ปิด Virtual Environment เมื่อใช้เสร็จ

powershell
deactivate
STEP 11

ครั้งถัดไปที่จะใช้งาน

ไม่ต้องสร้าง venv ใหม่ แค่เปิด PowerShell แล้วรัน:

powershell
cd Desktop\jupyter-project
.\venv\Scripts\Activate.ps1
jupyter notebook
STEP 12

ปัญหาที่พบบ่อย

ปัญหาวิธีแก้
"running scripts is disabled"รันคำสั่งใน STEP 03 อีกครั้ง
python ไม่รู้จักติดตั้ง Python ใหม่ + ติ๊ก Add to PATH
jupyter ไม่รู้จักเช็คว่า (venv) ขึ้นหน้า prompt หรือยัง
พอร์ต 8888 ถูกใช้งานjupyter notebook --port 8889
import numpy/pandas ไม่ผ่านรัน STEP 07 อีกครั้งขณะ (venv) active อยู่
ติดตั้งไลบรารีเพิ่มpip install ชื่อไลบรารี

รับไฟล์ข้อมูลสอบ
แล้วเริ่มทำข้อสอบ

ขั้นตอนสำหรับนักเรียน: ดาวน์โหลดไฟล์ ZIP ที่อาจารย์แจก แตกไฟล์ แล้วเปิดใน Jupyter Notebook (เครื่องตัวเอง) เพื่อเริ่มทำข้อสอบ

EXAM 01

ดาวน์โหลดไฟล์ข้อมูล

ดาวน์โหลดไฟล์ Exam_Data.zip จากลิงก์ที่อาจารย์แจกในห้องสอบ (Google Classroom หรือ Google Drive)

EXAM 02

แตกไฟล์ ZIP ไว้ในโฟลเดอร์โปรเจกต์

  • ย้ายไฟล์ Exam_Data.zip ไปไว้ในโฟลเดอร์ Desktop\jupyter-project (โฟลเดอร์ที่สร้างไว้ตอนติดตั้ง Jupyter ด้านบน)
  • คลิกขวาที่ไฟล์ → เลือก Extract All (หรือดับเบิลคลิกถ้าเครื่องมี 7-Zip / WinRAR)
  • ยืนยันให้แตกไฟล์ลงในโฟลเดอร์เดียวกัน จะได้ RegistrationName.xlsx และ StudentScores.xlsx อยู่ข้างๆ ไฟล์ notebook
💡 ต้องแตกไฟล์ไว้ ในโฟลเดอร์เดียวกับ notebook ไม่งั้นตอนอ่านไฟล์ด้วย pd.read_excel() จะหาไฟล์ไม่เจอ
EXAM 03

เปิด Jupyter Notebook แล้วเริ่มทำข้อสอบ

เปิด PowerShell แล้วรัน (ตามที่ตั้งค่าไว้ในส่วนที่ 1):

powershell
cd Desktop\jupyter-project
.\venv\Scripts\Activate.ps1
jupyter notebook

เมื่อเบราว์เซอร์เปิดหน้า Jupyter ขึ้นมา จะเห็นไฟล์ RegistrationName.xlsx และ StudentScores.xlsx อยู่ในรายการไฟล์ (มาจากที่แตก ZIP ไว้ในโฟลเดอร์เดียวกัน)

  • กด New → Python 3 (ipykernel) เพื่อสร้าง notebook ใหม่สำหรับทำข้อสอบ
  • ตั้งชื่อไฟล์ตามที่อาจารย์กำหนด (คลิกที่ชื่อ "Untitled" ด้านบนเพื่อเปลี่ยนชื่อ)
  • อ่านไฟล์ทั้งสองด้วย pandas ในเซลล์แรก แล้วเริ่มทำข้อสอบตามโจทย์ได้เลย
📌 อย่าลืมย้ำ: คำตอบทุกข้อต้องพร้อมเลขข้อกำกับในไฟล์ และหมั่นกด Ctrl + S บันทึกไฟล์ระหว่างทำข้อสอบเป็นระยะ เพราะ Jupyter local ไม่ได้บันทึกอัตโนมัติขึ้นคลาวด์เหมือน Colab
ส่วนนี้สำหรับนักเรียนใช้ตอนวันสอบ แยกจากขั้นตอนติดตั้ง Jupyter ด้านบน
